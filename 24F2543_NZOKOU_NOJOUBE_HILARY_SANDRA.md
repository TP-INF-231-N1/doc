ce projet avait pour but de pratiquer et daprofondir la programationen language c a travers des programmes dont jai envoye trois programmes differents traitant des notions fondamentales des tableaux de tri et d operation sur les vecteurs
description mediane d un tableau
ce programme demande a l utilisateur d entrer les elements d un tableau puis calcule et affiche la mediane 
on commence par tier le tableau pour identifier correctement la mediane si le tableau est paire on a deux elements medianes si c est impaire on a un element mediane
description du tri d un tableau
programme permattant de trier les elements d un tableau en ordre croissant on utilise un algorithme de tri simple le tableau initial saisi par l utilisateurest reorganise puis affiche trier
description du produit vectoriel
ce programme effectue le produit vectoriel de deux vecteurs a trois dimensions le programme affiche le vecteur resultant du produit vectoriel
en conclusion
ce devoir nous a permis de implementer un algorithme de tri apliquer une formule mathematique dans un programme renforcer la logique algorithmique et la rigueurdu codage en groupe
